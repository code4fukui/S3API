# S3API

S3バケットの操作を提供するAPIです。

## 機能
- S3バケットへのファイルアップロード
- S3バケットからのファイルダウンロード
- S3バケットのファイル一覧取得
- S3バケットのファイル削除

## 必要環境
- Deno 1.20以降

## 使い方
1. `s3.secret.yml`にAWSのアクセス情報を設定します。
2. `S3.js`をインポートして`S3`クラスのインスタンスを作成し、`init()`メソッドを呼び出します。
3. `put()`、`get()`、`list()`、`del()`メソッドを使ってS3操作を行えます。
